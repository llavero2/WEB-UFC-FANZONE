# UFC Fanzone 🥊

¡Bienvenido a **UFC Fanzone**! Este es un proyecto web estático diseñado para apasionados de las artes marciales mixtas. La página ofrece una experiencia completa para explorar la historia de la organización, conocer a los peleadores legendarios y revisar los próximos eventos.

## 🚀 Características
El sitio web consta de 5 secciones principales:
- **Historia:** Un recorrido desde los inicios de la UFC en 1993 hasta la actualidad.
- **Templo de la Fama:** Perfiles detallados de guerreros como Ilia Topuria, Conor McGregor y Jon Jones.
- **Próximos Eventos:** Carteleras actualizadas de los eventos más importantes (UFC 300, 301, etc.).
- **Tienda Oficial:** Catálogo de productos como guantillas y equipamiento.
- **Contacto:** Formulario integrado para consultas de los fans.

## 🛠️ Tecnologías utilizadas
Para el desarrollo de este proyecto se han utilizado:
- **HTML5:** Estructura semántica para todas las páginas.
- **CSS3:** Diseño personalizado con una estética oscura y acentos en rojo oficial de la UFC.
- **Diseño Responsivo:** Optimizado para una visualización clara en distintos dispositivos.

## 📦 Estructura del Proyecto
El proyecto está organizado de la siguiente manera:
- `index.html` (Página principal de Historia)
- `pagina 2.html` (Sección de Peleadores)
- `pagina 3.html` (Sección de Eventos)
- `pagina 4.html` (Tienda)
- `pagina 5.html` (Formulario de Contacto)
- `CSS UFC.css` (Hoja de estilos global)
